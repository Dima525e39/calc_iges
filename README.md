# IGES Cut Calculator MVP

Прототип Windows-программы для расчета периметра резки, количества врезок и стоимости по толщине материала.

Версия рассчитана на 3D IGES труб: при наличии Open Cascade программа читает файл как CAD-модель и автоматически выбирает режим для круглой или профильной трубы.
Интерфейс приложения сделан на русском языке.

- Круглая труба: программа ищет наружную цилиндрическую поверхность и считает граничные контуры на ней.
- Профильная труба: программа ищет наружные плоские боковые грани прямоугольного/квадратного профиля и считает резы/отверстия на них.
- Произвольные ребра 3D-модели не суммируются: в расчет идет только найденный контур резки.

## Что уже умеет

- Загружает файлы `.igs` и `.iges`.
- Считает периметр резки в метрах.
- Оценивает количество врезок как число отдельных контуров реза.
- Хранит таблицу толщин: цена за метр реза и цена за врезку.
- Пересчитывает стоимость при смене толщины.
- Читает 3D IGES через Open Cascade (`cadquery-ocp`).
- Если Open Cascade недоступен, использует запасной простой парсер IGES-кривых.

В режиме Open Cascade поддержка зависит от CAD-ядра: surfaces, shells, solids, trimmed surfaces и 3D-кривые читаются через полноценный импортер IGES. Запасной простой парсер поддерживает только `100` circular arc, `102` composite curve, `106` copious data/polyline, `110` line, `126` rational B-spline curve.

## Как считается круглая труба

1. Программа открывает IGES через Open Cascade.
2. Ищет цилиндрические грани.
3. Выбирает цилиндрические грани с самым большим радиусом как наружную поверхность трубы.
4. Суммирует длину граничных контуров этой наружной поверхности: торцы, отверстия и вырезы.
5. Количество врезок = количество отдельных контуров на наружной поверхности.

Малые цилиндры отверстий и внутренняя стенка трубы игнорируются по радиусу.

## Как считается профильная труба

1. Программа открывает IGES через Open Cascade.
2. Определяет длинную ось трубы по габаритам модели.
3. Ищет наружные плоские боковые грани на габаритной оболочке профиля.
4. Суммирует граничные ребра этих граней, которые не являются продольными углами профиля.
5. Количество врезок = количество отдельных контуров реза на наружных гранях.

Этот режим рассчитан на квадратные и прямоугольные профильные трубы. Если цилиндрических и профильных наружных граней нет, программа не суммирует все ребра модели, чтобы не завышать периметр резки.

## Самый простой вариант для пользователя

Для конечного пользователя лучше собрать один из двух файлов:

- `dist\IGESCutCalculator.exe` - портативная программа, можно просто отправить файл.
- `dist\IGESCutCalculatorSetup.exe` - установщик, который добавляет ярлыки и ставит программу в `Program Files`.

Собирать нужно на Windows.

## Запуск из исходников

На Windows установите Python 3.11 x64 и зависимости:

```powershell
python -m pip install -r requirements-build.txt
```

Запустите:

```powershell
python app.py
```

## Сборка портативного `.exe`

Откройте папку проекта на Windows и запустите:

```powershell
build_windows.bat
```

Скрипт сам установит PyInstaller и соберет:

```text
dist\IGESCutCalculator.exe
```

Этот файл можно передать пользователю без установки Python.

Важно: из-за CAD-ядра Open Cascade файл `.exe` может быть заметно больше обычной маленькой программы. Это нормально для приложения, которое читает 3D CAD.

## Автоматическая сборка через GitHub

В проект добавлен workflow:

```text
.github\workflows\build-windows.yml
```

Если загрузить проект в GitHub, сборку можно запустить во вкладке `Actions` -> `Build Windows EXE` -> `Run workflow`. После завершения в artifacts появится:

```text
IGESCutCalculator-windows
```

Внутри будет готовый:

```text
IGESCutCalculator.exe
```

## Сборка установщика `.exe`

1. Установите Inno Setup 6: <https://jrsoftware.org/isinfo.php>
2. Запустите:

```powershell
build_installer_windows.bat
```

Готовый установщик:

```text
dist\IGESCutCalculatorSetup.exe
```

## Настройки цен

При первом запуске используются цены по умолчанию. После нажатия `Сохранить настройки` настройки сохраняются в профиле пользователя:

```text
%APPDATA%\IGES Cut Calculator\settings.json
```

Формула:

```text
стоимость = периметр_резки_м * цена_за_метр + количество_врезок * цена_за_врезку
```

## Важное ограничение

Даже с Open Cascade первые реальные файлы нужно сверить с CAM/производственной системой. Особенно важны случаи:

- Профильная труба повернута в IGES под произвольным углом относительно осей координат.
- Профильная труба не квадратная/прямоугольная.
- В IGES есть несколько труб в одном файле.
- Рез идет по внутренней поверхности или по средней линии стенки, а не по наружной.
- Торцевой рез не должен считаться как врезка, потому что станок начинает с края.
- 3D-модель содержит фаски, гибы или неплоские поверхности.
- В IGES есть несколько деталей в одном файле.
- Файл содержит не листовую деталь, а полноценную объемную модель.

Для следующей версии стоит добавить предпросмотр 3D и настройки трубы: круглая/профильная, считать ли торцевые резы как врезки, считать по наружной поверхности или по средней линии стенки.
